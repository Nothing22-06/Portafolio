Temario

Unidad I: Desarrollo web fronted

Capacidad: Desarrollar aplicaciones web frontend de páginas simples SPA de acuerdo a la especificación de documentos.

Semana N°1 : Exposición de sílabo y Prueba de diagnóstico 6%

Objetivo : Conocer el funcionamiento básico de la web, herramientas y roles en el desarrollo web.

Semana N°2 :Construir una página con Lenguaje Marcado de Hipertexto HTML y Lenguaje Marcado Extensible XML 12%

Objetivo : Construir páginas con HTML y CSS, aplicando conceptos de SEO y diseño responsivo.

Semana N°3 : Desarrollar una página web Front-End con Librería CSS Bootstrap 18%

Objetivo : Utilizar librerías CSS para crear componentes visuales avanzados en el frontend.

Semana N°4 : Desarrollar una página web Front-End con Javascript y Typescript 24%

Objetivo : Aplicar JavaScript y Canvas para agregar lógica e interactividad a páginas web.

Semana N°5 : Desarrollar aplicación FrontEnd con Framework JS 28%

Objetivo : Introducirse al desarrollo con frameworks JS, estructurando componentes y utilizando props y estilos.

Semana N°6 : Desarrollar aplicaciones con Eventos, Condicionales y Bucles con Framework JS 30%

Objetivo : Manejar eventos y consumir datos de APIs utilizando promesas y renderizado condicional.

Semana N°7 : Uso de Hooks 36%

Objetivo : Aplicar hooks como useState, useEffect y useContext para gestionar estados y efectos en componentes React.

Reflexiones de las clases

Semana 1

Aprendí los fundamentos de la web y la importancia de cada rol en el desarrollo. Lo hice explorando el entorno y aplicando lo visto en clase.

Semana 2

Comprendí cómo estructurar correctamente una página con HTML y mejorar su diseño con CSS. Practiqué bastante con ejercicios y laboratorios.

Semana 3

Aprendí a usar Bootstrap para construir componentes rápidos y visualmente atractivos. Me apoyé en la documentación oficial y videos.

Semana 4

Me introduje a JavaScript y TypeScript, desarrollando interacciones simples. Entendí mejor practicando con animaciones en Canvas.

Semana 5

Conocí los frameworks JS y cómo estructurar una aplicación con componentes. Fue retador pero muy gratificante.

Semana 6

Aprendí a manejar eventos, condicionales y consumir APIs. Experimentar en vivo fue esencial para asimilar el conocimiento.

Semana 7

Usé hooks como useState y useEffect. Logré gestionar estado y efectos de forma más limpia y entendí el flujo de datos en React.

Fin del Curso

Consolidé los conocimientos adquiridos y ahora me siento más segura desarrollando aplicaciones web modernas.
